# LearnexaAI
Frontend for an AI-powered study assistant that transforms PDFs into summaries, notes, flashcards, quizzes, and an interactive learning experience.

## Features

- 📄 AI Summary
- 📝 Smart Notes
- 🧠 Interactive Flashcards
- ❓ AI-Generated Quiz 
- 🤖 AI Doubt Solver

## Tech Stack

- HTML
- CSS
- JavaScript
- n8n (Backend Automation)
- Telegram Bot
